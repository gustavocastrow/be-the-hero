# Be the hero

<b>A aplicação desenvolvida foi a "Be the hero" que visa conectar ONGs com pessoas que apoiam suas causas.</b>


<ul>
  <b>Para criação da aplicação foi utilizado:</b>
  <li>ReactJS</li>
  <li>React Native</li>
  <li>NodeJS</li>
  <li>Express</li>
  <li>Expo</li>
  <li>SQLite</li>
  <li>Axiso</li>
  <li>Cors</li>
  <li>Knex</li>
  <li>Jest</li>
 </ul>
